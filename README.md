# bank-loan-risk-sql-analysis
SQL project analyzing bank loan data to identify loan performance, repayment behavior, and financial profit and loss insights.

Project Overview
This project analyzes bank loan data using SQL to understand loan performance, borrower repayment behavior, and financial risk. The analysis helps identify profitable lending segments and detect areas where banks may experience financial loss.
________________________________________
Tools Used
•	SQL Server
•	Excel
•	Power BI (for visualization)
________________________________________
Key Business Questions
•	How many total loan applications were processed?
•	What percentage of borrowers successfully repaid their loans?
•	How much loan amount was funded by the bank?
•	What portion of loans resulted in financial loss?
•	What is the total profit generated from loan repayments?
________________________________________
Key Insights
• Analyzed 38K+ loan records to evaluate borrower repayment behavior.
• Identified good loan percentage vs bad loan percentage to measure credit risk.
• Calculated total funded amount, repayment amount, and bank profit/loss.
• Determined loan segments contributing to higher default risk.
________________________________________
Key SQL Analysis
Major queries included:
•	Loan applications analysis
•	Month-to-date loan performance
•	Loan repayment percentage
•	Good vs bad loan classification
•	Bank profit and loss calculation
________________________________________
Business Impact
The analysis helps banks:
•	Identify high-risk borrowers
•	Improve loan approval strategies
•	Reduce default risk
•	Increase overall profitability
